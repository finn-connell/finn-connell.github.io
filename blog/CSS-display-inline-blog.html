Utilising the Display Attribute

display: inline vs. display: inline-block

Inline elements can be styled as blocks within their container elements using the CSS property "display." Inline elements have their display property set as "display= inline;" if no property is specified. By setting the display property as "display= inline-block;" you can have an inline HTML object such as a span behave almost as if it has its own container element. This means that you can set height and width parameters for it, and have it wrap other content in its container to boot. This opens up many opportunities to tinker with your layouts within containers.
